## [1.0.0] - 2026-04-28

### 新增
- 🎨 首次发布
- 💰 显示剩余话费（大字体突出显示）
- 📊 显示剩余/已用流量（GB）
- 📞 显示剩余/已用语音（分钟）
- 💵 显示已用话费
- 📅 显示套餐周期进度
- 🌈 渐变色彩系统（蓝/青/紫/绿/橙/粉）
- ✨ 平滑动画效果
  - 卡片入场动画
  - 图标呼吸脉冲
  - 进度条加载动画
  - 悬停微交互
- 📱 响应式布局（适配移动端）
- 🔌 支持传感器集成
- ⚙️ 支持手动配置
- 🎯 进度条可视化
- 🌙 深色主题设计

### 技术细节
- 基于 Lit Element 的自定义元素
- 支持 Home Assistant 2023.0.0 及以上版本
- HACS 兼容
- 完整的配置选项

- **主版本号 (Major)**: 不兼容的 API 更改
- **次版本号 (Minor)**: 向后兼容的功能新增
- **修订号 (Patch)**: 向后兼容的问题修复
